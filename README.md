# Infinite Resurrection Barbarian

Welcome to the **Infinite Resurrection Barbarian** project! 

This is a Phaser-based game where you explore dungeons, fight enemies, and progress through levels.

## How to Play

1. Clone this repository.
2. Open `index.html` in your browser.
3. Use the keyboard to move and interact (refer to in-game instructions).

## Technical Details

- **Engine**: [Phaser 3](https://phaser.io/)
- **Assets**: Custom 2D pixel art and audio.

## GitHub Pages Deployment

To play online, visit the [GitHub Pages](https://pupologic.github.io/Infinite-Resurrection-Barbarian/) link.
